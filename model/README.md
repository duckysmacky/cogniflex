# Рабочая область моделей

Эта директория предназначена для исследований, обучения, оценки моделей, а так же
содержит утилиты, необходимые для работы с данными и моделями.

## Структура

notebooks/ - .ipynb файлы содержащие исследование данных, обучение моделей, а также
оптимизацию моделей для продакшена
utils/ - инструменты, используемые в процессе обучения, тестирования моделей; некоторые из них являются составной частью утилил внутри *prod_utils*
prod_utils/ - утилиты для работы с моделями и данными в продакшене, среди них:
workflow_model/ - тестирование моделей
