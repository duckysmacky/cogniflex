# Cogniflex

Detecting AI-generated content in real time

## About

Cogniflex is an open platform designed to help people automatically identify and warn about AI-generated content on the internet. It analyzes digital content and estimates the probability that it was generated by artificial intelligence.

The system focuses on **three main types of content**:

- Text - articles, posts, comments, and other written material
- Images - pictures that may have been generated using diffusion models or other AI systems
- Videos - videos that may have been also generated by AI

Instead of making absolute claims, Cogniflex provides probabilistic assessments, helping users interpret information more critically.

> [!NOTE]
> Cogniflex is currently in an early development phase

## Features

### Real-time content analysis

The main feature of the project is that the system is designed to work **directly in the browser as an extension** - just turn it on and the program will automatically start analyzing the content on the page, highlighting the AI-generated content right on the screen.

This allows to quickly understand whether the content you are viewing was likely generated by artificial intelligence **without needing to switch tabs or copy-pasting any content**.

### Manual analysis tools

We also provide ability for users to perform static (manual) analysis. If you don't want for the extension to analyze the content on the page automatically, you can manually paste text, images and videos to the static analysis interfaces, which is present in the extension, as well as can be also accessed on our website.

### Multi-method detection

Instead of relying on a single model, Cogniflex combines multiple approaches, including:

- machine learning models
- statistical analysis
- structural content patterns
- heuristic algorithms

This hybrid approach improves reliability and flexibility.
